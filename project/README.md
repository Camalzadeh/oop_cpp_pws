# Chessboard Project - Comprehensive C++ Implementation

Welcome to the terminal-based Chess engine. This project is a complete implementation of a Chess game, developed as a final requirement for the Object-Oriented Programming course.

## 🚀 Quick Start

### Compilation
In the `project` directory, run:
```bash
g++ *.cpp -o chess
```
*(Alternatively, if `make` is installed, just type `make`).*

### Running the Game
```bash
./chess
```

## ✅ Features Implemented

- **Standard Chess Rules**: Supports all standard piece movements.
- **Check Detection**: Prevents illegal moves and notifies of checks.
- **Game End States**: Detects Checkmate and Stalemate.
- **Pawn Promotion**: Detects when a pawn reaches the last rank.
- **English Refactor**: All code, documentation, and interface terms have been translated to English for clarity.
